# Changelog

## 0.1.0 — May 2026

**Trust Gateway**
- Initial public specification published
- Ozone labeler live at `labeler.kevara.app`
- Label vocabulary live: `kevara-verified`, `kevara-pro`, `kevara-gander`, `kevara-eurosky`, `kevara-recruiter`
- Public label query endpoint live: `com.atproto.label.queryLabels`
- WebSocket label subscription live: `com.atproto.label.subscribeLabels`
- Subscription-level revocation live (Pro lapse negates `kevara-pro` automatically)

**Lexicon Reference**
- Initial reference published covering all active `id.sifa.*` and `is.kevara.*` schemas
